# 🏦 Bank API

API RESTful de um sistema bancário simples, desenvolvida com **FastAPI**, com funcionalidades de autenticação, gerenciamento de contas e operações financeiras.

---

## 🚀 Objetivo do Projeto

Este projeto foi criado para simular o funcionamento básico de um sistema bancário real, permitindo:

- Cadastro de usuários
- Autenticação (login)
- Criação de contas bancárias
- Operações financeiras (depósito e saque)
- Consulta de saldo

---

## 🧠 Tecnologias Utilizadas

- **Python**
- **FastAPI**
- **Uvicorn**
- **Render (Deploy)**
- **GitHub**

---

## ⚙️ Funcionalidades

### 🔐 Autenticação

- Registro de usuários
- Login com validação de credenciais

### 🏦 Contas Bancárias

- Criação de conta vinculada a um usuário
- Cada conta possui ID único e saldo

### 💰 Transações

- Depósito em conta
- Saque com validação de saldo
- Consulta de todas as contas
